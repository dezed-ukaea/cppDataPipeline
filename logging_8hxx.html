Detailed Description

File contains pointers for creating global logger during session.

Author
K. Zarebski (UKAEA)
Date
2021-05-06

This file should be included in all FDP header files as it allows all debug/error information to be output to the same logger and hence the same terminal session. It uses the SPDLog library.
